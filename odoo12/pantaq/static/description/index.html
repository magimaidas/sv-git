PANTAQ

(Module created for Pantaq to handle Procurement process)


Initial Configuration


  • Create some procurement team ( Sales >> Configuration >> Sales Teams(Team Category as Procurement)) & assign Users to the Team

 


Enquires

  • Open Purchase or CRM module for enquires

 


 

  • Create Enquiry & Covert to RFQ


 


RFQs (Request For Quotation)

  • Map Product & Supplier to Create RFQ

 

Based on the number of vendors mapped to the products, the number of RFQs will be created for submission

  • Update the RFQ and send

 


 

Note: Once RFQ is sent, need to update the status via Response received button.


 

  • Approve or Reject RFQ

 


 

  • Compare price from vendors and decide to approve or reject

 

(Click on Purchase >> RFQ Product price compare. Select Enquiry # and proceed)


 

  • RFQ compare and approve in the list view


 


Internal Quotations

  • Create Internal Quotation from Enquiry

 


 

  • Update margin, discount in line items

 


 


Customer Quote( i.e. Sale Order )

  • Convert Internal Quotation to Customer Quote

 


 

  • Update profit in line items and Confirm sale

 


 

  • Profit analysis ( We can find the profit analysis estimation below the order lines )
