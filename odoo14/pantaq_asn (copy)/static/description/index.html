Warehouse Goods Receiving Flow – ASN



Purpose:

This module helps in Goods Receiving Flow with ASN inspection. ASN ( Advance Shipping Note) is used to make the receiver be aware of the goods to be received in advance. With the help of this module, you can inspect the goods and verify its condition before it is added to the actual stock. If found of damaged, you can return the particular item or make it to scap. 


Steps:

  1. Enquiry to RFQ

  2. RFQ to Purchase Order

  3. Purchase Order to ASN

  4. ASN to Stock

  5. Stock Pickings

    1. Return

    2. Scrap
